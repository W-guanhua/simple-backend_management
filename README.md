# 构建打包方式
+ yarn 安装依赖
+ yarn serve 编译运行测试环境
+ yarn build 编译运行生产环境

# 使用开发流程
1. yarn add-project(如果有新增项目时执行，否则可忽略)
2. yarn add-apiList(配置好scripts/add-apiList/config.js后调用)
3. yarn create-project(配置好scripts/create-project/config.js后调用)
4. 修改类型及继续开发内容
